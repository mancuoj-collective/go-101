## 方法和接口

- Go 没有类，但是可以为类型定义方法
- 方法是一类带特殊的接受者参数的函数
- 接收者的类型定义和方法声明必须在同一包内

```go
type Vertex struct {
	X, Y float64
}

// 正常函数
func Abs(v Vertex) float64 {
	return math.Sqrt(v.X*v.X + v.Y*v.Y)
}

// 方法
func (v Vertex) Abs() float64 {
	return math.Sqrt(v.X*v.X + v.Y*v.Y)
}

func main() {
	v := Vertex{3, 4}
	fmt.Println(Abs(v)) // 调用函数
  fmt.Println(v.Abs()) // 调用方法
}
```
